# عرض فهيم — Amad hackathon deck (Arabic, RTL)

`faheem-deck-ar.pptx` is the judge-facing deck. 16 slides, Arabic-only, fully RTL,
built on the Faheem brand system (navy `#061F52` / emerald `#07966F`, tokens from
`app/globals.css`) and following the material flow of
`context/Lunar Team - Agentic AI Financial Platform (Faheem).pdf`.

## Fonts — REQUIRED on the presenting machine

The deck uses **IBM Plex Sans Arabic** (Regular / Medium / SemiBold / Bold) and
**Amiri** (thanks slide only). Install them before presenting or PowerPoint will
substitute and wreck the layout:

- IBM Plex Sans Arabic: <https://fonts.google.com/specimen/IBM+Plex+Sans+Arabic>
- Amiri: <https://fonts.google.com/specimen/Amiri>

Download → select all `.ttf` → right-click → Install. (Already installed on this
machine under `~/.fonts/faheem-deck/`.)

## Regenerating

The deck is generated by `src/build-deck.mjs` (pptxgenjs) from `src/assets/`
(brand glyph renders, hackathon/sponsor logos extracted from the source PDF,
recolored lucide icons, app screenshot).

```bash
cd presentation/src
npm install          # pptxgenjs + sharp + lucide-static (dev-only, NOT app deps)
node assets.mjs      # only if you need to regenerate assets/
node build-deck.mjs  # writes faheem-deck-ar.pptx here; copy up one level
```
